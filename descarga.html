Descarga

El codigo fuente de gugel es distribuido mediante https. Enlace de descarga: Descargar Gugel.

Para poder abrir el archivo, es necesario usar alguna herramienta para extraer archivos. En caso de no tener ninguna aplicación para extraer archivos, debajo tendrá enlaces hacia las paginas que distribuyan este tipo de software.


Herramientas

Para descargar Winrar, dirijase a: Sitio oficial winrar.

Para descargar WinZip, dirijase a: Sitio oficial WinZip.

Para descargar 7-Zip, dirijase a: Sitio oficial 7-Zip.
